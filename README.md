# Swift Internship

This repository tracks my progress toward an iOS interview.

## Daily Plan
- Study core Swift structures: classes, functions, data structures (arrays, sets, dictionaries, etc.), including time and space complexity analysis.
- Implement fundamental algorithms in Swift: sorting, hashing, searching.
- Solve 2–3 LeetCode problems.
- Learn specific SwiftUI syntax blocks (views, modifiers, state management).

## Weekly Plan
- Plan and build one small project at the start of each week, applying the knowledge gained so far.
- Study one actively used iOS framework (SwiftUI, UIKit, Core Data).

## Repository Structure
- Separate branches for each week to mark progress milestones.
- Daily commit reports summarizing what I learned, implemented, or started.
